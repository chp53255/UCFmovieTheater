# UCFmovieTheater

A full-stack web application built with the MERN stack (MongoDB, Express.js, React, Node.js) for CIS 4004 — Spring 2026.

## Overview

A movie theater ticketing platform that allows users to browse movies and showings, book tickets, and manage their reservations. Administrators can manage movies, theaters, showings, and view all bookings.

## Tech Stack

- **MongoDB** — Document database (local instance via `mongodb://localhost:27017`)
- **Express.js** — Backend web framework and REST API
- **React** — Frontend single-page application
- **Node.js** — JavaScript runtime

## Notes
- The application runs entirely on localhost — no deployment required.
- MongoDB must be running locally before starting the backend server.
- Passwords are stored as plaintext for this project (no hashing required).
